$project
========

$project :: This python3.6 program parses the output data from a quantum chemical calculation and stores the data in python data structures. 
Currently, it can also dump this information in the form of input files of CHAMP program.


Look how easy it is to use:

    import pychamp
    # Get your stuff done
    python input.py
